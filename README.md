🚨 **Hyperlocal Emergency Help Network**

This project is a web-based emergency assistance system designed to connect people in critical situations with nearby volunteers quickly and efficiently. The primary goal is to reduce response time during emergencies by providing a simple and reliable one-click solution to access help.

The system supports two types of users: **Users** and **Volunteers**. Users can register and log in to access an emergency dashboard, while volunteers register themselves to provide help to people in need within their locality.

Once logged in, users are provided with a dedicated dashboard that includes an **SOS button**. In case of an emergency, pressing this button instantly connects the user to a volunteer. If internet connectivity is available, the system can also share the user’s live location. In situations without internet access, the system still functions by directly initiating a phone call using the mobile network.

The project is developed using **HTML, CSS, and JavaScript**, and uses browser **LocalStorage** to store user and volunteer data. This makes the system lightweight, easy to deploy, and suitable as a working prototype without requiring a backend server.
